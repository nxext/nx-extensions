# @nxext/react:init

This schematic installs the following deendencies:

```
react
react-dom
```

## Usage

```
nx g @nxext/react:init
```

## Options

### --unitTestRunner

Default: `jest`

Possible values: jest, none

Type: `enum`

Adds the specified unit test runner.

### --skipFormat

Default: `false`

Type: `boolean`
