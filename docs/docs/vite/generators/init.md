# @nxext/vite:init

This schematic installs the following deendency:

```
vite
```

## Usage

```
nx g @nxext/vite:init
```

## Options

### --unitTestRunner

Default: `jest`

Possible values: jest, none

Type: `enum`

Adds the specified unit test runner.

### --skipFormat

Default: `false`

Type: `boolean`
