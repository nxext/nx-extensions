Generates a Svelte component

## Usage

```
nx g @nxext/svelte:c my-app
```

or

```
nx g @nxext/svelte:component my-app
```

## Options

### --project

Alias(es): p

Type: `string`

Project where the component is generated

### --unitTestRunner

Default: `jest`

Possible values: jest, none

Type: `enum`

Adds the specified unit test runner.

### --skipFormat

Default: `false`

Type: `boolean`
