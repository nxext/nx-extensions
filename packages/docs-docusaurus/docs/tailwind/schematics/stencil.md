This schematic installs the following packages:

```
tailwindcss
@stencil/postcss
autoprefixer
nanocss
```

Into the stencil.config.ts will be added the postcss plugin config for the tailwind usage.

## Usage

```
nx g @nxext/tailwind:stencil my-app
```

## Options

### --skipFormat

Default: `false`

Type: `boolean`
