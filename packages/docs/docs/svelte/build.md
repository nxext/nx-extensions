## Build

Build your project:

```
nx build my-app
```

Supported flags are:

| Parameter | Type | Default | Description |
| ------------ | ------ | ---------------------------------------------------------------------------------------- | ---------------------------- |
| --dev | bool | false | Is dev build |
| --watch | bool | false | Start watchmode |
| --serve | bool | false | Serve on dev webserver |
| --open | bool | false | Open in browser |

## Serve

Serve with:

```
nx serve my-app
```

Supported flags are:

| Parameter | Type    | Default | Description  |
| --------- | ------- | ------- | ------------ |
| --dev     | bool    | false   | Is dev build |
| --open    | boolean | false   |              |
